# BMI Calculator

A sleek, cyber-inspired Body Mass Index (BMI) calculator themed around the *CSI: Cyber* TV show. Users can input their height and weight to calculate their BMI and receive a categorization of their health status, all within a futuristic, neon-lit interface.

---

## Features

- User-friendly form for height and weight input
- Real-time BMI calculation with validation
- Cyberpunk-inspired visual theme with neon colors and dark backgrounds
- Clear display of BMI result and health category
- Designed with HTML, CSS, and JavaScript for easy customization and learning

## How it Works

1. **Input Data:** Enter height in centimeters and weight in kilograms.
2. **Calculate BMI:** Click the "Calculate BMI" button.
3. **View Results:** The BMI value and category are displayed with a neon cyber aesthetic.

---

## Technologies Used

- **HTML5** for structure
- **CSS3** for styling with neon and dark theme
- **JavaScript** for calculations and interactivity
- **Google Fonts** (Orbitron) for a futuristic font

---

## Setup Instructions

1. **Clone or Download** this repository, or copy the code into your preferred IDE.
2. **Open `index.html`** in your browser to see the BMI calculator in action.
3. **Customize** the styles or logic as desired.

---

## Customization Tips

- Change the color scheme in CSS to match your preferred cyber palette.
- Add additional features like BMI history or health tips.
- Integrate charts or graphs for visualizing BMI progress.
